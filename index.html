Операційна система GNU/Linux

Коротка інформація

GNU/Linux — це вільна та відкрита операційна система, яка базується на ядрі Linux та інструментах GNU. Вона відома своєю стабільністю, безпекою та гнучкістю. Часто використовується на серверах, робочих станціях, мобільних пристроях (Android) і вбудованих системах.

Приклади команд Linux CLI

pwd        # показати поточну директорію
echo "Привіт"   # вивести повідомлення
ls         # перелік файлів у директорії
cd /home   # перейти до директорії /home
touch file.txt   # створити новий файл
mkdir my_folder  # створити нову папку
cp file1.txt copy.txt   # копіювати файл
mv oldname.txt newname.txt   # перейменувати/перемістити файл
rm file.txt     # видалити файл
history         # показати історію команд

BASH-скрипт авторизації

#!/bin/bash

correct_user="admin"
correct_pass="12345"

read -p "Введіть логін: " username
read -s -p "Введіть пароль: " password
echo ""

if [[ "$username" == "$correct_user" && "$password" == "$correct_pass" ]]; then
  echo "Авторизація успішна. Ласкаво просимо, $username!"
else
  echo "Помилка: неправильний логін або пароль."
fi
